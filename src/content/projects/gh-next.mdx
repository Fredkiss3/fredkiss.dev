<LocalizedContent locale='en'>
A GitHub clone built with Next.js, utilizing the new App Router paradigm and server components.
The goal was to recreate GitHub's UI and core functionality from the ground up. 
Key features include a custom-built issue search page, from database design to UI, closely mirroring GitHub's backend structure. 
Additionally, the UI was enhanced with a more effective keyword highlighting system in the search input.
</LocalizedContent>

<LocalizedContent locale='fr'>
Un clone de GitHub réalisé avec Next.js, utilisant le nouveau paradigme App Router et les server components.  
L’objectif était de recréer l’interface et les fonctionnalités principales de GitHub depuis zéro.  
Les points clés incluent une page de recherche d’issues sur mesure, couvrant la conception de la base de données jusqu’à l’interface, reprenant fidèlement la structure backend de GitHub.  
De plus, l’interface a été améliorée avec un système de mise en valeur des mots-clés plus efficace dans le champ de recherche.
</LocalizedContent>
