# My portfolio

This is my portfolio, made with [astro](https://astro.build) and deployed on [ZaneOps](https://github.com/zane-ops/zane-ops).

# Requirements

- Node >= v16.6.2
- [PNPM](https://pnpm.io/installation) >= v8.15.9
